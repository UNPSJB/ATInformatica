Interfaz de usuario
===================

WebSaS fue pensado como un sistema de gestión de órdenes de trabajo para los servicios
de reparación ofrecidos por AT-Informática. El sistema permite gestionar:

* Clientes
* Técnicos
* Tipos de servicio
* Rubros
* Tareas y tarifas
* Productos y reservas
* Órdenes de Trabajo
