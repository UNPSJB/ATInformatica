En esta pantalla se presenta un asistente paso a paso que le permite abrir una nueva orden de trabajo.

El asistente le permite crear un nuevo cliente o equipo, pero no puede registrar nuevos técnicos, tipos de servicio, rubros o tareas.

Para concluir con la creación de una OT, deberá haber completado los datos necesarios en cada paso (por ejemplo, seleccionar un cliente).
